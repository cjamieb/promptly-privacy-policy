Privacy Policy

Last updated: 1/15/2026

Overview

This extension is designed to help users optimize prompts submitted to AI tools in order to reduce response time, computational cost, and environmental impact. Our mission is to promote more sustainable AI usage by providing transparency into estimated energy consumption and emissions.

We take user privacy seriously and collect only the data necessary to provide these features.

Information We Collect

1. Account Information (via Google Sign-In)

If you choose to sign in using Google, we collect:

  • A unique Google account identifier (provided by Google OAuth)
  • Basic profile information necessary to associate usage data with your account (e.g., email address)

We do not access your Google password and do not collect contacts or unrelated Google account data.

2. AI Usage Metrics (User-Generated Data)

The extension collects user-provided and computed metrics, including:

  • Estimated number of AI tokens used
  • Estimated energy consumption
  • Estimated device battery usage (“charges”)
  • Estimated carbon emissions related to AI usage

These values are calculated by the extension based on user interactions and publicly available estimation models. We do not intercept private AI conversations unless explicitly provided by the user as part of optimization.

3. Local Extension Data

We store the following using Chrome’s local storage:

  • User preferences and settings
  • Aggregated usage statistics
  • Extension state data

How We Use This Information

Collected data is used solely to:

  • Display sustainability metrics to the user
  • Track personal usage trends over time
  • Improve prompt optimization features
  • Support user accounts and sign-in functionality

We do not use data for advertising, profiling, or resale.

Data Storage & Security

  • Data is stored securely using Chrome’s built-in storage APIs and/or secure backend services.
  • Authentication is handled via Google OAuth using chrome.identity.
  • Access to user data is limited to authorized systems only.
  • We implement reasonable technical and organizational safeguards to protect user information.

Data Sharing

We do not sell, rent, or share user data with third parties.

Data is only shared:

  • When required to authenticate via Google Sign-In
  • If legally required to comply with applicable laws

User Control & Data Deletion

Users may:

  • Sign out of the extension at any time
  • Clear locally stored data by removing the extension
  • Request deletion of account-associated data by contacting us at:
    [your contact email]

Permissions Explanation

  • storage: Used to save user preferences and usage metrics locally so data persists across sessions.
  • identity: Used to allow users to sign in with Google and associate sustainability metrics with their account.

No additional permissions are requested or used.

Children’s Privacy

This extension is not intended for use by children under the age of 13. We do not knowingly collect personal data from children.

Changes to This Policy

We may update this Privacy Policy as the extension evolves. Changes will be reflected with an updated “Last updated” date.
